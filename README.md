# Blockchain Staking DApp Scrum Project

## Overview

This repository demonstrates the application of Scrum and Agile methodologies in managing the development of a Blockchain Staking DApp.

The project was organized using Jira and showcases backlog management, sprint planning, sprint execution, sprint reviews, retrospectives, and Agile reporting.

---

## My Role

Scrum Master

---

## Responsibilities

- Facilitated Sprint Planning sessions
- Managed the Product Backlog
- Defined Sprint Goals
- Monitored Sprint Progress
- Facilitated Sprint Reviews
- Conducted Sprint Retrospectives
- Managed stakeholder communication
- Tracked team performance using Jira reports

---

## Project Objective

Develop a decentralized staking application that enables users to connect wallets, stake tokens, calculate rewards, and interact with the application through a user-friendly dashboard.

---
